### 隐私声明
最后更新：2025年12月
DeepSwim 致力于保护你的个人信息和隐私。本隐私声明说明我们如何收集、使用和保护你的数据。

#### 数据收集
我们收集以下信息：
- 游泳训练数据（距离、时间、心率等）
- 个人健康信息（身高、体重、出生日期）
- 应用使用偏好设置
 
#### 数据使用你的数据用于：
- 展示你的游泳训练历史和统计数据
- 提供个性化的训练分析和见解
- 改善应用功能和用户体验
  
#### 数据存储
- 所有数据仅存储在你的设备本地
- 我们不会将你的数据上传到任何服务器
- 你可以随时删除本地存储的所有数据
- 数据通过iOS 系统加密保护
  
#### HealthKit 集成
- 我们通过 Apple HealthKit 访问你的健康数据
- 仅在你授权后才能访问相关数据
-  你可以随时在系统设置中撤销权限
- 我们遵守 Apple 的健康数据使用政策
  
#### 你的权利
- 查看和管理你的所有数据
- 随时删除本地存储的数据
- 控制应用对 HealthKit 的访问权限
- 随时撤销已授予的权限
  
#### 联系我们
如有任何隐私相关问题或疑虑，请通过应用内的用户反馈功能联系我们。
